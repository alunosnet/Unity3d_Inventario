# Unity3d_Inventario
Sistema de inventário para o Unity 3D com funcionalidade de persistência dos items

Um simples sistema de inventário para o Unity 3D com as seguintes funcionalidades:
- adicionar;
- verificar se existe;
- utilizar;
- guardar inventário;
- carregar inventário;
- mostrar items do inventário;
